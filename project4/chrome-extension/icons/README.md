# Chrome Extension Icons

This directory contains icons for the AI Email Response Generator Chrome extension.

## Icon Requirements

- **icon16.png**: 16x16 pixels - Used in the toolbar and favicon
- **icon48.png**: 48x48 pixels - Used in the extensions management page
- **icon128.png**: 128x128 pixels - Used in the Chrome Web Store

## Design Guidelines

The icons should follow these design principles:
- Clean, modern design
- AI/email themed (robot, envelope, or combination)
- Consistent color scheme with the extension UI
- High contrast for visibility
- Professional appearance suitable for business use

## Recommended Design

A simple envelope with a robot/AI brain icon would be appropriate:
- Blue/purple gradient background (matching extension theme)
- White envelope icon
- Small robot/AI symbol overlay
- Clean, minimalist design

## Implementation

You can create these icons using:
- Adobe Illustrator or Figma for vector design
- Canva for quick online design
- Online icon generators
- Or hire a designer for professional results
